# Marks Sorter

**Author:** Sudipta Kundu  

**Description:**  
Marks Sorter is a frontend tool that allows users to input student names and marks, sort them, and export the results as Word or PDF files. It is designed to help teachers and students create attendance/marks sheets quickly and efficiently.  

**Technologies Used:**  
- HTML  
- CSS  
- JavaScript  

**Instructions to Run:**  
1. Download or clone the repository.  
2. Open the `index.html` file in any web browser.  
3. Input student names and marks.  
4. Click the buttons to sort the list and export as Word or PDF.

**Notes:**
